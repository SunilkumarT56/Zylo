# Code of Conduct

This project follows a simple rule: be respectful and professional.

## Our Standards

We expect all contributors to:

- Be respectful and considerate
- Accept constructive feedback
- Focus on technical discussions
- Treat others with professionalism

Unacceptable behavior includes:

- Harassment or discrimination
- Personal attacks or insults
- Trolling or inflammatory comments
- Publishing private information without consent

## Enforcement

Project maintainers have the right to:

- Remove inappropriate content
- Temporarily or permanently ban contributors
- Reject contributions that violate this code

## Scope

This code of conduct applies to:

- GitHub issues
- Pull requests
- Discussions
- Any project-related communication

## Reporting

If you encounter unacceptable behavior, report it privately to the
maintainers.
